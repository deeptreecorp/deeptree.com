# DeepTree Website

This is the static website for DeepTree, hosted at https://deeptree.com

## Project Structure

- `index.html` - Main landing page
- `logo.png` - DeepTree logo
- `CNAME` - GitHub Pages custom domain configuration

## Hosting

The site is hosted on GitHub Pages with a custom domain (deeptree.com).

## Analytics

The site uses Plausible Analytics for privacy-friendly visitor tracking.

## Development

This is a simple static HTML site. To make changes:
1. Edit `index.html` directly
2. Commit and push to `main` branch
3. GitHub Pages will automatically deploy
