# DeepTree Website

The official website for DeepTree, live at https://deeptree.com

## Project Structure

```
deeptree.com/
├── index.html    # Main landing page
├── logo.png      # DeepTree logo
├── CNAME         # Custom domain config
├── CLAUDE.md     # Claude Code context
└── AGENTS.md     # AI agent guidelines
```

## Development

This is a static HTML site with no build step. To develop locally:

1. Clone the repository
2. Open `index.html` in your browser
3. Edit and refresh to see changes

## Deployment

The site is hosted on GitHub Pages. Push to the `main` branch to deploy.
